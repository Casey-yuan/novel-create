# 《都市奇探手记》世界观设定

## 一、时代背景

### 时间设定
- **年代**：当代（2020年代）
- **季节**：故事跨越春夏秋冬，展现都市四季变迁
- **时间流速**：正常时间流速，案件之间有时间间隔

### 社会背景
- **经济环境**：经济稳步发展的现代都市，贫富差距存在但不极端
- **科技水平**：当代科技水平，智能手机、监控、DNA检测等技术普及
- **社会氛围**：整体安定，但都市阴影处仍有犯罪滋生

---

## 二、地理设定

### 主要城市：江城

#### 城市概况
- **定位**： fictional 的现代化大都市，人口约800万
- **特点**：
  - 临江而建，江河穿城而过
  - 老城区与新开发区并存
  - 商业繁华与市井生活交织
  - 四季分明，气候宜人

#### 核心区域

| 区域名称 | 特点 | 案件关联 |
|---------|------|---------|
| **滨江新区** | 高端商务区，摩天大楼林立 | 商业犯罪、豪门恩怨 |
| **老城区** | 历史街区，巷弄纵横 | 传统案件、民间纠纷 |
| **大学城** | 高校聚集，青春气息 | 校园案件、学术纠纷 |
| **工业园区** | 工厂企业集中 | 经济犯罪、劳资纠纷 |
| **城郊结合部** | 城乡交界，管理薄弱 | 连环案件、隐藏罪恶 |

#### 标志性地点

1. **奇探事务所**
   - 位置：老城区一条安静的街道
   - 外观：三层老式洋楼，外观古朴，内部现代化装修
   - 一层：接待区、会客室
   - 二层：办公区、档案室、会议室
   - 三层：主角私人空间

2. **江城警察局**
   - 位置：市中心
   - 作用：警方合作、案件交接

3. **江城大学**
   - 位置：大学城核心
   - 作用：学术案件、年轻角色活动地

4. **滨江商业街**
   - 位置：滨江新区
   - 作用：商业案件、都市生活场景

## 三、社会规则

### 法律体系
- 现代中国法律体系
- 警方依法办案，程序正义
- 私家侦探在法律边缘游走，但不触犯法律底线

### 侦探行业规则
- 私家侦探需依法注册
- 调查手段需合法合规
- 与警方保持合作关系
- 不得干涉司法公正

### 社会阶层
- **上层**：企业家、富豪、名流
- **中层**：白领、专业人士、小企业主
- **下层**：普通工人、服务业人员、边缘人群

---

## 四、核心设定

### 侦探事务所运营模式

#### 业务范围
1. **民事调查**：失踪人口、婚姻调查、财产追踪
2. **商业调查**：商业间谍、知识产权、背景调查
3. **刑事案件协助**：警方委托、疑难案件、冷案重启
4. **特殊委托**：寻物、安保咨询、风险评估

#### 收费模式
- 按案件复杂度收费
- 部分案件采用"成功收费"模式
- 公益案件酌情减免

### 案件类型体系

#### 单元案件分类

| 类型 | 特点 | 示例 |
|------|------|------|
| **密室推理** | 封闭空间犯罪 | 密室杀人、不可能犯罪 |
| **不在场证明** | 时间诡计 | 时间差杀人、不在场证明破解 |
| **身份诡计** | 人物关系谜题 | 替身、身份互换、人格分裂 |
| **毒杀案件** | 毒物诡计 | 毒杀、药物诡计 |
| **密室逃脱** | 空间诡计 | 密室制造、逃脱手法 |
| **心理悬疑** | 心理博弈 | 心理操控、催眠犯罪 |
| **连环案件** | 模式犯罪 | 连环杀手、模仿犯罪 |
| **冷案重启** | 陈年旧案 | 多年未破案件重新调查 |

## 六、文化元素

### 饮食文化
- 江城特色小吃：热干面、豆皮、鸭脖
- 各类餐馆：高档餐厅、街边小店、夜市

- 传统节日：春节、中秋、端午

### 娱乐文化
- 电影院、KTV、酒吧
- 公园、广场、步行街
- 网络娱乐、短视频平台

## 七、世界观核心主题

### 主题一：正义与真相
- 每个案件都追求真相大白
- 正义可能迟到，但不会缺席
- 法律是维护正义的最后防线

### 主题二：人性复杂
- 每个罪犯都有犯罪动机
- 善与恶的边界有时模糊
- 理解人性，但不纵容罪恶

### 主题三：都市百态
- 展现都市生活的多样性
- 不同阶层的生活状态
- 都市人的喜怒哀乐

### 主题四：智慧与勇气
- 推理是智慧的较量
- 面对危险需要勇气
- 团队合作的力量

## 八、风格基调

### 整体风格
- **轻松幽默**：不沉重，有笑料
- **推理严谨**：逻辑严密，公平竞技
- **人物鲜活**：角色个性鲜明
- **节奏明快**：不拖沓，推进迅速

### 氛围营造
- 悬疑但不恐怖
- 紧张但有喘息
- 严肃但不失幽默
- 真实但不阴暗

---

## 九、创作原则

### 推理原则
1. **公平原则**：所有线索提前展示，读者可与侦探公平竞争
2. **逻辑原则**：推理过程严密，无逻辑漏洞
3. **意外原则**：结局出乎意料但情理之中
4. **多样原则**：案件类型丰富，手法多变

### 人物原则
1. **成长原则**：主角和配角都有成长弧线
2. **立体原则**：人物不脸谱化，有优点有缺点
3. **互动原则**：人物关系推动剧情
4. **记忆原则**：重要人物会重复出现

### 价值原则
1. **正义原则**：弘扬正义，鞭挞罪恶
2. **法治原则**：尊重法律，程序正义
3. **人文原则**：关注人性，理解多元
4. **积极原则**：整体基调积极向上
