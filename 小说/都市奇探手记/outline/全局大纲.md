# 《都市奇探手记》全局大纲（调整后）

## 一、作品概述

### 基本信息
- **书名**：《都市奇探手记》
- **题材**：刑侦悬疑 / 轻松推理
- **风格**：单元剧模式，类似《神探狄仁杰》
- **篇幅**：中篇，预计12-16万字
- **卷数**：2卷
- **章节**：60章
- **每章字数**：2000-3000字

### 核心卖点
1. **单元剧模式**：每个案件独立完整，适合碎片化阅读
2. **轻松风格**：悬疑但不恐怖，幽默但不轻浮
3. **推理严谨**：本格推理元素，公平竞技
4. **人物鲜活**：主角团性格鲜明，互动有趣
5. **都市背景**：贴近生活，案件类型多样

## 五、调整说明

### 主要调整

1. **总章节从80章压缩到60章**
   - 节奏更快，避免拖沓
   - 更容易保持高质量

2. **开篇从3章压缩到1章**
   - 第001章快速进入主线
   - 立即抓住读者注意力

3. **案件数量从8个减少到6个**
   - 删除原案四（连环盗窃）
   - 每个案件更饱满，推理更充分

4. **父亲案件提前**
   - 从第二卷中期提前到第一卷末尾
   - 增加悬念，提升追读动力

5. **第二卷结构调整**
   - 从40章压缩到30章
   - 从4个案件减少到3个案件
   - 主线案件占比增加

### 预期效果

1. **开篇更吸引人**：1章快速进入，节奏紧凑
2. **案件更精品**：6个精品案件比8个平庸案件更有吸引力
3. **主线更清晰**：父亲案件贯穿全篇，悬念更强
4. **节奏更紧凑**：60章结构更紧凑，阅读体验更好
