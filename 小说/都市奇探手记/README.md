**【故事梗概】**

十五年前，陆铭的父亲在调查一桩密室杀人案时殉职。

十五年后，同样的密室，同样的手法，同样的曼陀罗标记——再次出现。

"你父亲不是自杀。"

陆铭盯着墓碑，左手腕的机械表发出"咔哒"声。

那是父亲留下的唯一遗物，也是他思考时的习惯。

从被所有人嘲笑的"废物侦探"，到让罪犯闻风丧胆的"都市奇探"。

陆铭用智商碾压一切，揭开一个个离奇案件背后的真相。

**密室杀人、失踪谜案、毒杀疑云、冷案重启……**

每个案件都是一场智慧的较量，每一次破案都是对正义的坚守。

而当所有案件串联起来，一个惊人的真相浮出水面——

十五年前的悲剧，竟然是一个巨大阴谋的开始……
