# 《逐出宗门：我激活了万古大帝传承》世界观设定

## 一、世界概述

### 1.1 世界名称
**天玄大陆** - 一个以修仙为尊的玄幻世界

### 1.2 世界结构
天玄大陆分为五大域：
- **东域**：主角所在区域，宗门林立，灵气充沛
- **西域**：魔道势力盘踞，环境险恶
- **南域**：妖族领地，万妖横行
- **北域**：冰雪覆盖，上古遗迹众多
- **中州**：大陆中心，最强宗门和家族聚集

### 1.3 修炼体系

#### 境界划分（由低到高）
| 境界 | 特点 | 寿元 |
|------|------|------|
| 炼气期 | 引气入体，淬炼肉身 | 100年 |
| 筑基期 | 筑就道基，正式踏入修仙路 | 200年 |
| 金丹期 | 凝结金丹，法力大增 | 500年 |
| 元婴期 | 元婴出窍，神通初显 | 1000年 |
| 化神期 | 神识大成，可御空飞行 | 3000年 |
| 合体期 | 元神与肉身合一 | 5000年 |
| 大乘期 | 接近飞升，法力无边 | 10000年 |
| 渡劫期 | 渡天劫，成则飞升 | 20000年 |
| 大帝境 | 传说中的境界，万古无敌 | 永生 |

#### 战力体系
- 每个境界分九重小境界
- 同境界内，功法、法宝、战斗经验决定胜负
- 跨境界战斗极为困难，除非有逆天功法或法宝

### 1.4 宗门体系

#### 宗门等级
| 等级 | 说明 | 最强者修为 |
|------|------|-----------|
| 九品 | 最低级宗门 | 筑基期 |
| 八品 | 小型宗门 | 金丹期 |
| 七品 | 中型宗门 | 元婴期 |
| 六品 | 大型宗门 | 化神期 |
| 五品 | 顶级宗门 | 合体期 |
| 四品 | 超级宗门 | 大乘期 |
| 三品 | 圣地 | 渡劫期 |
| 二品 | 不朽传承 | 大帝境 |
| 一品 | 万古帝宗 | 多位大帝 |

#### 主角所在宗门：青云宗
- **等级**：六品宗门
- **位置**：东域青州
- **宗主**：李青云（化神期巅峰）
- **特色**：剑修为主，在东域有一定声望

## 二、核心设定

### 2.1 万古大帝传承

#### 传承来历
- **万古大帝**：十万年前横扫天玄大陆的绝世强者
- **成就**：统一五大域，建立万古帝宗，后神秘消失
- **传承**：包含功法《万古帝经》、帝器、丹方、阵法等

#### 传承激活条件
1. 被逐出宗门，心生绝望
2. 血脉与传承契合
3. 在特定地点（万古大帝陨落之地附近）

#### 传承内容
1. **《万古帝经》**：大帝级功法，修炼速度是普通功法的百倍
2. **帝器"万古剑"**：可成长性法宝，随主人境界提升而变强
3. **大帝记忆碎片**：包含万古大帝的战斗经验、炼丹术、阵法等
4. **神秘空间**：可种植灵药、存放物品、时间流速不同

### 2.2 关键地点

#### 青云宗
- 主角被逐之地
- 位于东域青州青云山脉
- 有外门、内门、核心弟子之分

#### 黑风岭
- 主角被逐后流落之地
- 妖兽横行，危险重重
- 万古大帝传承隐藏于此

#### 天玄城
- 东域最大城池
- 商业繁华，情报中心
- 主角崛起后建立势力的基地

## 三、势力分布

### 3.1 正道势力
- **青云宗**：主角原宗门，六品
- **天剑宗**：东域最强剑修宗门，五品
- **丹鼎阁**：炼丹师圣地，五品
- **万佛寺**：佛门势力，四品

### 3.2 魔道势力
- **血魔宗**：魔道魁首，四品
- **幽冥教**：擅长暗杀，五品
- **合欢宗**：双修门派，六品

### 3.3 中立势力
- **天机阁**：贩卖情报，神秘强大
- **万宝楼**：商业联盟，富可敌国
- **散修联盟**：散修组织，人数众多

## 五、故事时间线

### 背景时间
- **十万年前**：万古大帝横扫天下，建立万古帝宗
- **五万年前**：万古大帝神秘消失，万古帝宗衰落
- **一万年前**：万古帝宗彻底覆灭，传承断绝
- **千年前**：青云宗建立，逐渐崛起

### 故事时间
- **故事开始**：主角被逐出青云宗，激活传承
- **故事跨度**：约3年（从被逐到成为大帝）
