# 《逐出宗门：我激活了万古大帝传承》人物档案

## 一、主角

### 林渊

#### 基本信息
| 项目 | 内容 |
|------|------|
| 姓名 | 林渊 |
| 性别 | 男 |
| 年龄 | 18岁（故事开始时） |
| 身份 | 青云宗外门弟子 → 被逐 → 万古大帝传承者 |
| 灵根 | 废品灵根（表面）/ 混沌灵根（激活传承后） |
| 性格 | 隐忍坚韧、重情重义、杀伐果断、护短 |

#### 外貌描写
- 身高：180cm
- 体型：修长挺拔，肌肉匀称
- 面容：剑眉星目，轮廓分明，气质冷峻
- 特征：眼神深邃如渊，嘴角常带一丝若有若无的冷笑
- 服饰：前期穿青云宗外门灰袍，后期一袭黑衣，腰悬万古剑

#### 人物背景
- 出身：东域林家旁系，父母双亡
- 经历：10岁进入青云宗，8年苦修仍在炼气三层
- 被逐原因：被师兄陷害， accused of 偷窃宗门灵药
- 转折点：被逐后在黑风岭濒死，激活万古大帝传承

#### 能力设定
**前期（被逐后-筑基期）**
- 功法：《万古帝经》第一层
- 法宝：万古剑（帝器，初始状态）
- 特殊能力：大帝记忆碎片（战斗经验）

**中期（金丹期-化神期）**
- 功法：《万古帝经》中层
- 法宝：万古剑（觉醒部分威能）、神秘空间
- 特殊能力：炼丹术、阵法、越级战斗

**后期（大乘期-大帝境）**
- 功法：《万古帝经》大圆满
- 法宝：万古剑（完全觉醒）、多件帝器
- 特殊能力：万古领域、时间法则、空间法则

#### 成长弧线
1. **屈辱期**：被逐宗门，人人欺辱，绝望濒死
2. **崛起期**：激活传承，快速成长，初露锋芒
3. **复仇期**：回归打脸，清算旧账，名震东域
4. **巅峰期**：成为大帝，揭开万古之谜，成就传说

#### 经典台词
- "今日你逐我出门，他日我让你跪迎！"
- "我林渊一生，不跪天不跪地，只跪父母恩师！"
- "辱我者，百倍奉还；欺我者，永世不得超生！"

#### 基本信息
| 项目 | 内容 |
|------|------|
| 姓名 | 苏清雪 |
| 性别 | 女 |
| 年龄 | 17岁（故事开始时） |
| 身份 | 青云宗宗主之女，天骄弟子 |
| 灵根 | 天灵根（冰属性） |
| 性格 | 外冷内热、善良正直、敢爱敢恨 |

#### 外貌描写
- 身高：168cm
- 体型：纤细窈窕，如冰雪般清冷
- 面容：肌肤胜雪，眉目如画，气质出尘
- 特征：眼眸如寒星，长发如瀑，常着白衣
- 气质：清冷如仙子，令人不敢亵渎

#### 人物背景
- 出身：青云宗宗主李青云独女
- 地位：宗门第一天才，众星捧月
- 与主角关系：唯一在主角被逐时出言相助的人
- 感情线：被主角的坚韧吸引，后生死相随

#### 能力设定
- 功法：《玄冰真诀》（青云宗顶级功法）
- 法宝：冰魄剑（上品灵器）
- 特殊能力：冰系神通、治愈之术

#### 人物作用
- 感情线核心，与主角相互扶持
- 连接主角与青云宗的纽带
- 后期成为主角最信任的伙伴
