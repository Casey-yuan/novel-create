# 市场雷达与数据分析

Fiction Crafter 内置市场分析功能，帮助作品更好地适配市场需求。

---

## 市场雷达功能

### 热门题材监测

实时追踪当前市场热门题材：

| 都市修仙 | 🔥🔥🔥🔥🔥 | ↑ | 高 | 差异化定位 |
| 重生逆袭 | 🔥🔥🔥🔥 | → | 高 | 创新切入点 |

## 智能建议

### 基于数据的建议

根据作品数据提供智能建议：

```
【更新建议】
根据您的更新数据，建议：
- 保持当前更新频率（日均XXXX字）
- 最佳发布时间：晚上8-10点
- 建议每周至少更新5章

【质量建议】
根据质量分析，建议关注：
- 穿帮问题较多，建议加强记忆系统使用
- 第15-20章节奏较慢，注意爽点分布
- 主角成长曲线平稳，可在第25章设置突破

【市场建议】
根据市场分析，建议：
- 当前题材热度上升，可加大更新量
- 建议在第30章设置大高潮，提升讨论度
- 可考虑增加感情线，扩大读者群
```
