# 书名查重功能

在确定小说名字之前，必须先进行书名查重，确保不与已有小说重名，特别是在番茄小说平台。

---

## 查重规则

### 强制要求
**在确定小说名字之前，必须先进行书名查重**

### 查重范围
1. **番茄小说**（重点）- 国内最大免费小说平台
2. **起点中文网** - 付费小说龙头
3. **晋江文学城** - 女频重要平台
4. **飞卢小说网** - 脑洞文聚集地
5. **七猫小说** - 免费阅读平台
6. **百度/谷歌搜索** - 全网查重

### 查重标准
- **完全匹配**：书名完全相同，必须放弃
- **高度相似**：仅差1-2个字，建议放弃
- **部分匹配**：核心词相同，需要差异化
- **无匹配**：可以使用

---

## 查重流程

```
用户提供初步书名想法
    ↓
生成3-5个候选书名
    ↓
对每个候选书名进行查重
    ↓
番茄小说平台（重点）
    ↓
其他小说平台
    ↓
全网搜索
    ↓
汇总查重结果
    ↓
推荐最佳书名
    ↓
用户确认最终书名
    ↓
继续后续流程
```

## 书名生成策略

### 如果查重发现重名

#### 策略1：添加差异化前缀/后缀
- 原书名：《都市修仙》
- 修改后：《都市修仙：从签到开始》、《重生之都市修仙》

#### 策略2：替换同义词
- 原书名：《最强系统》
- 修改后：《无敌系统》、《至尊系统》

#### 策略3：添加具体元素
- 原书名：《修仙大帝》
- 修改后：《修仙大帝：开局签到十万年》

#### 策略4：改变句式结构
- 原书名：《我成了反派》
- 修改后：《穿书后我成了反派》、《我，反派，打钱》

### 书名优化公式

#### 公式1：核心词+差异化标签
```
《[核心词]：[差异化标签]》

示例：
- 《都市修仙：我的系统能签到》
- 《重生之我是大反派：开局退婚女主》
```

#### 公式2：身份+核心冲突
```
《[主角身份] + [核心冲突/行动]》

示例：
- 《废柴少爷的逆袭之路》
- 《女配她只想躺平》
```

#### 公式3：数字+核心设定
```
《[数字] + [核心设定]》

示例：
- 《999次重生后我无敌了》
- 《第108次攻略失败》
```

#### 公式4：场景+悬念
```
《[场景] + [悬念]》

示例：
- 《我在殡仪馆当化妆师》
- 《每天醒来都在不同人床上》
```

## 书名规范

### 字数限制
- **番茄小说**：≤15个字符（含中英文、数字、符号）
- **推荐长度**：8-12个字，便于记忆和传播

### 字符规范
- 支持：中英文、阿拉伯数字
- 支持中文符号：叹号、逗号、问号、中括号、冒号
- 不支持：英文标点符号

### 禁用内容
- 涉及政治敏感
- 涉及色情暴力
- 涉及封建迷信
- 侵犯他人权益
- 违反公序良俗

---

## 书名优化建议

### 好书名的标准
1. **好记**：朗朗上口，便于传播
2. **吸睛**：有吸引力，让人想点开
3. **贴合**：与内容相符，不挂羊头卖狗肉
4. **独特**：有差异化，避免同质化
5. **直白**：网文书名要直白，不要文艺腔

### 爆款书名元素
- **身份标签**：总裁、王妃、学霸、反派
- **关系设定**：闪婚、契约、团宠、退婚
- **情绪痛点**：后悔、虐渣、撩、逆袭
- **风格基调**：甜宠、爽文、救赎、黑暗
- **数字元素**：99次、第108次、三年
- **反差萌**：修仙+火锅、总裁+扫地机器人

## 注意事项

1. **书名不涉及版权**：书名本身一般不涉及版权问题，但重名会影响辨识度
2. **番茄平台重点**：番茄小说对书名查重较严格，重点关注
3. **多书名实验**：番茄支持多书名实验，可以准备多个备选
4. **书名可修改**：大部分平台支持上架前修改书名，但会影响推广
5. **提前查重**：建议在创作前就确定好书名，避免中途修改
