# 篇幅规划与题材适配

根据用户选择的篇幅和题材，提供专业的篇幅建议和题材适配分析。

---

## 篇幅规格定义

### 短篇
- **字数范围**：3万-10万字
- **章节数**：15-50章
- **卷数**：1卷（不分卷）
- **创作周期**：1-2个月
- **适合场景**：练笔、投稿、短篇比赛、IP改编试水

### 中篇
- **字数范围**：10万-30万字
- **章节数**：50-150章
- **卷数**：1-2卷
- **创作周期**：2-4个月
- **适合场景**：网文试水、特定题材、短篇系列

### 长篇
- **字数范围**：30万-100万字
- **章节数**：150-500章
- **卷数**：3-5卷
- **创作周期**：6-12个月
- **适合场景**：主流网文、商业写作、IP开发

### 超长篇
- **字数范围**：100万字以上
- **章节数**：500章以上
- **卷数**：5卷以上
- **创作周期**：1年以上
- **适合场景**：超级IP、系列作品、特定题材

## 篇幅选择流程

### 第一步：询问用户篇幅需求

```
Planner Agent：

在开始创作之前，我需要了解您的篇幅需求：

【请选择您想创作的篇幅规格】

1. **短篇** (3-10万字)
   - 适合：练笔、投稿、短篇比赛、快速完本
   - 周期：1-2个月

2. **中篇** (10-30万字)
   - 适合：网文试水、特定题材、短篇系列
   - 周期：2-4个月

3. **长篇** (30-100万字)
   - 适合：主流网文、商业写作、IP开发
   - 周期：6-12个月

4. **超长篇** (100万字以上)
   - 适合：超级IP、系列作品、特定题材
   - 周期：1年以上

请输入您的选择（1-4）：
```

### 第二步：询问题材

```
Planner Agent：

请告诉我您想创作的题材类型：

【主题材】
1. 玄幻
2. 仙侠
3. 都市
4. 科幻
5. 悬疑
6. 历史
7. 游戏
8. 体育
9. 言情
10. 武侠
11. 其他

请输入题材编号：

【题材细分】（根据主题材显示对应细分）
例如选择"玄幻"后显示：
- 东方玄幻
- 异世大陆
- 高武世界
- 低武江湖
- 洪荒封神

请输入细分题材：
```

### 第三步：根据题材给出篇幅建议

```
Planner Agent：

【篇幅适配分析】

根据您选择的题材 **[题材名]**，我为您提供以下分析：

**您的选择**：长篇 (50万字)

**专业建议**：
- **推荐篇幅**：长篇 (40-80万字)
- **匹配度**：⭐⭐⭐⭐⭐ (非常适合)
- **原因分析**：
  - [题材特点1]
  - [题材特点2]
  - [题材特点3]

**风险提示**：
- [风险1及应对建议]
- [风险2及应对建议]

**市场参考**：
- 该题材成功案例：《[书名1]》(XX万字)、《[书名2]》(XX万字)
- 平均完本字数：XX万字
- 读者偏好：倾向于[篇幅类型]

**建议调整**（如需要）：
如果您的时间/精力有限，可以考虑调整为：
- 中篇 (20-30万字)：保留核心设定，精简支线
- 短篇 (10-15万字)：聚焦一个完整故事线

请确认您的篇幅选择：
1. 维持原选择：[原篇幅]
2. 调整为：[建议篇幅]
3. 自定义：[输入字数]
```

### 第四步：生成基于篇幅的规划

```
Planner Agent：

【基于篇幅的创作规划】

确认篇幅：**[篇幅]** ([字数范围])

**自动调整规划**：
- 卷数：[N]卷
- 每卷章节数：[N]章
- 总章节数：[N]章
- 每章字数：2000-3000字
- 预计总字数：[N]万字
- 建议创作周期：[周期]

**爽点分布调整**：
- 根据篇幅调整爽点密度
- 短篇：每3-5章一个爽点
- 中篇：每5-8章一个爽点
- 长篇：每8-15章一个爽点

**剧情节奏调整**：
- 短篇：快节奏，直入主题
- 中篇：中等节奏，有起有伏
- 长篇：慢热开局，逐步升级

现在开始制定详细规划...
```

## 题材篇幅禁忌

### 绝对禁忌（容易导致烂文）

| 题材 | 禁忌篇幅 | 原因 | 后果 |
|------|---------|------|------|
| 校园青春 | 超长篇(100万+) | 时间跨度有限 | 强行续写，脱离校园 |
| 侦探推理 | 超长篇(100万+) | 案件有始有终 | 多个案件拼接，质量下降 |
| 甜宠文 | 长篇(50万+) | 甜度过长会腻 | 后期为甜而甜，读者流失 |
| 短篇科幻 | 超长篇 | 科幻设定难以长期维持 | 后期变成玄幻 |
| 体育竞技 | 超长篇 | 赛事周期固定 | 重复比赛，缺乏新意 |

### 相对禁忌（需要特别注意）

| 题材 | 需谨慎的篇幅 | 注意事项 |
|------|-------------|---------|
| 悬疑恐怖 | 长篇以上 | 恐怖氛围难以长期维持，需要不断创新 |
| 单女主言情 | 超长篇 | 感情线单一，后期容易拖沓 |
| 国术无双 | 长篇以上 | 国术天花板有限，后期容易脱离现实 |
| 电子竞技 | 长篇以上 | 赛事模式固定，容易重复 |
| 历史传记 | 超长篇 | 容易变成流水账 |

## 篇幅规划检查清单

### 选择篇幅时检查
- [ ] 是否询问了用户篇幅需求？
- [ ] 是否了解了用户题材选择？
- [ ] 是否根据题材给出了专业建议？
- [ ] 是否说明了推荐原因和风险？
- [ ] 是否提供了调整选项？
- [ ] 用户是否确认了最终篇幅？

### 规划阶段检查
- [ ] 篇幅与题材是否匹配？
- [ ] 卷数设置是否合理？
- [ ] 每卷章节数是否符合篇幅要求？
- [ ] 爽点密度是否适配篇幅？
- [ ] 剧情节奏是否适配篇幅？
- [ ] 是否有强行拉长或压缩的痕迹？

### 创作阶段检查
- [ ] 是否按照篇幅规划执行？
- [ ] 是否有注水或赶工的情况？
- [ ] 剧情进度是否符合篇幅预期？
- [ ] 是否需要调整篇幅？
